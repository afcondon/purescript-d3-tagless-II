CodeSnippet

#codeSnippet 

codeSnippet :: forall w i. String -> String -> HTML w i

Render a code snippet with Prism.js syntax highlighting, filename, and line numbers

Usage: codeSnippet "mySnippet" "purescript"

The snippet will be extracted from your source files by adding this comment: SNIPPET: mySnippet src/path/to/file.purs 10-25

This shows the source filename and line numbers matching the original file. Note: Call triggerPrismHighlighting in your component's Initialize action

#triggerPrismHighlighting 

triggerPrismHighlighting :: forall m. MonadEffect m => m Unit

Trigger Prism highlighting after code snippets are rendered Call this in your component's Initialize action

getStartLine :: String -> Int
